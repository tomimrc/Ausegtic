
# Grupo AuSegTIC – Sitio Web

Este proyecto replica y moderniza el sitio del **Grupo AuSegTIC (Grupo de Auditoría y Seguridad de Tecnologías de Información y Comunicaciones)** de la **Universidad Tecnológica Nacional, Facultad Regional Mendoza (UTN FRM)**.

El objetivo es **mantener íntegramente la información** del sitio original, pero con un **diseño nuevo y moderno**

---

## Estructura general del sitio

Páginas principales:

- `/` – Inicio
- `/objetivos`
- `/integrantes`
- `/docencia`
- `/proyectos`
- `/produccion-transferencia`

Todas las páginas comparten:

- **Nombre del sitio:** Grupo AuSegTIC  
- **Descripción corta:** Grupo de Auditoría y Seguridad de TIC  
- **Menú de navegación:** Inicio, Objetivos, Integrantes, Docencia, Proyectos, Producción/Transferencia

## Estilo y diseño

Este proyecto **no replica el estilo visual original** del sitio de UTN FRM.  
En su lugar, propone:

- Un diseño **moderno e institucional**, con:
  - Hero en la página de inicio.
  - Secciones bien separadas, con tipografías claras y espaciado generoso.
  - Componentes visuales tipo cards, tablas y acordeones para manejar gran volumen de contenido.
  - Cada sección de este README a **secciones** o **bloques** en las distintas páginas.
  - Los listados largos (publicaciones, transferencia, comités) a acordiones, tabs o listas segmentadas.

---

De esta forma se preserva **la información completa del sitio original** pero presentada en un diseño nuevo y más usable.
